# ImageStudy

Projeto desenvolvido em React.js consumindo a API do Pexels utilizando e aprofundando os seguintes conhecimentos:

- Componentes
- Styled Components
- Typescript
- Context
- Consumo de API
- Axios
- React Icons
- Responsividade
